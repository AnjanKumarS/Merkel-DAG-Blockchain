DAG Consensus Model

The DAG (Directed Acyclic Graph) consensus model is an alternative to traditional blockchain architectures that offers higher scalability. Instead of grouping transactions into blocks, each transaction can reference multiple previous transactions directly.

This implementation is inspired by the IOTA Tangle, where:

  • New transactions must approve 2+ existing transactions (tips)
  • Transactions accumulate weight as they're referenced by newer transactions
  • Tips are unconfirmed transactions waiting for approval
  • The system uses a randomized tip selection algorithm

Orange nodes in the visualization represent tips, while blue nodes represent confirmed transactions. Node size corresponds to cumulative weight.
